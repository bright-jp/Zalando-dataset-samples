# Zalando-dataset-samples

<h2>1001件のZalando商品からなるサンプルデータセット</h2>

![Zalando dataset header](https://github.com/luminati-io/Zalando-dataset-samples/blob/main/Zalando-datasets.png)

1000件を超えるレコードのZalandoデータセットサンプルです。データセットは <b>Bright Data API</b> を使用して抽出されました。

<h2>Zalandoデータセットに含まれるデータポイントの一部:</h2>

* ```domain```: 商品が属するドメインまたはカテゴリ
* ```country_code```: 商品に関連付けられた国コード
* ```url```: オンラインで商品の全詳細を確認するためのURLまたはリンク
* ```sku```: 在庫管理単位（Stock Keeping Unit）。商品の一意識別子
* ```condition```: 商品の状態
* ```gender```: 商品の対象性別
* ```product_name```: 商品名またはタイトル
* ```brand```: 商品のブランド
* ```description```: 商品の詳細説明
* ```manufacturer```: 商品の製造元または生産者
* ```badges```: 商品に関連付けられた特別なバッジまたはラベル
* ```initial_price```: 商品の初期（元）価格
* ```final_price```: 商品の最終（割引後）価格
* ```discount```: 商品に適用された割引率
* ```currency```: 価格が記載されている通貨
* ```inventory```: 利用可能な在庫に関する情報
* ```is_sale```: 現在商品がセール中かどうかを示します

ほかにも多数あります。

これは「Zalando products」データセットから派生したサンプルのサブセットであり、同データセットには <b>970万件以上のレコード</b> が含まれます。

利用可能なデータセットファイル形式: <b>JSON, NDJSON, JSON Lines, CSV, または Parquet。必要に応じて、ファイルは .gz に圧縮できます</b>。

データセット配信タイプのオプション: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>。

更新頻度: <b>1回のみ、毎日、毎週、毎月、四半期ごと、またはカスタム</b>。

抽出されたデータポイントに追加できるデータエンリッチメント: <b>リクエストに応じて。</b>

<b>[Zalandoデータセットの全文を取得](https://brightdata.jp/products/datasets/zalando)</b>。

<h2>Zalandoデータセットのユースケースとは?</h2>

<h3>1. 価格最適化</h3>
Zalandoの商品やカテゴリを競合他社のものと比較することで、価格戦略を策定し、ダイナミックプライシングモデルを実装します。

<h3>2. 商品在庫の最適化</h3>
在庫の不足箇所を特定し、特定商品の需要増を追跡し、Zalandoの商品ラインナップ内のトレンドアイテムを可視化します。

<h3>3. 市場戦略の最適化</h3>
Zalandoデータセットを使用して主要トレンドと顧客の嗜好を分析し、市場戦略を強化します。
